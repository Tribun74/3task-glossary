# 3task Glossary – Dictionary, Wiki & Knowledge Base

Create glossaries, dictionaries and knowledge bases using WordPress pages. Features A-Z navigation, automatic term linking, tooltips, and dark mode.

[![WordPress Plugin Version](https://img.shields.io/wordpress/plugin/v/3task-glossary)](https://wordpress.org/plugins/3task-glossary/)
[![WordPress Plugin Downloads](https://img.shields.io/wordpress/plugin/dt/3task-glossary)](https://wordpress.org/plugins/3task-glossary/)
[![License: GPL v2](https://img.shields.io/badge/License-GPL_v2-blue.svg)](https://www.gnu.org/licenses/gpl-2.0)

## Features

- A-Z alphabetical navigation
- Automatic term linking in posts & pages
- Tooltip previews on hover
- Uses WordPress pages (no custom database tables)
- Search functionality
- Dark mode support
- German translation included
- GDPR-friendly (no external services)
- SEO-optimized markup

## Installation

1. Upload `3task-glossary` to `/wp-content/plugins/`
2. Activate the plugin in WordPress
3. Go to **3task Glossary** in your admin menu
4. Configure and start adding terms

Or install directly from [WordPress.org](https://wordpress.org/plugins/3task-glossary/).
